Define matrices by suffix (f32, q31, q15) for use in test cases.

The rand1 and rand2 suffixes get their data from the same pool of random data, but their starting points differ by 1 element.

Makes available:

  • matrix_suffix_1x1_rand1/2
  • matrix_suffix_1x4_rand1/2
  • matrix_suffix_2x4_rand1/2
  • matrix_suffix_4x4_rand1/2
