◆ CFFT_FAMILY_DEFINE_TEST

#define CFFT_FAMILY_DEFINE_TEST (   fn_specifier,
  type_suffix,
  config_suffix,
  comparison_interface,
  output_type,
  ifft_flag 
)

Defines a test for the family of CFFT transforms.

The family of CFFT transforms includes:

  • arm_cfft_radix4_xxx
  • arm_cfft_radix2_xxx
  • arm_cfft_mag_xxx

Where xxx can be f32, q31, or q15.

Аргументы
fn_specifierAllowed values: radix4, radix2, mag.
type_suffixAllowed values: f32, q31, q15.
config_suffixUsed to differentiate test names based configuration (in this case whether the ifft_flag is set or not.)
comparison_interfaceMacro name used to compare reference and fut outputs.
output_tpeThe type of variable contained in the output (e.g. float32_t, uint32_t, etc).
ifft_flagDetermines whether the arm_cfft_instance_xxx is configured for an inverse FFT.
