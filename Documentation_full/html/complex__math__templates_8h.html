Compare reference and fut outputs using SNR.

The output_suffix specifies which output buffers to use for the comparison. An output_suffix of 'a' expands to the following buffers:

  • complex_math_output_f32_ref_a
  • complex_math_output_f32_fut_a
  • complex_math_output_ref_a
  • complex_math_output_fut_a
Заметки
The outputs are converted to float32_t before comparison.
